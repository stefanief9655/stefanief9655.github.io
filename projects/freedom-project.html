Freedom Project

The purpose of this project is to research a topic and take notes on the softwares and hardwares within it. With this information we then had to come up with an innovation and a purpose for it to serve or problem for it to help solve. With our information and new innovation we now needed to create a website to display this information. This website was then presented by me to judges at the SEP10 Expo. For this presentation I had to write a short speech to say called an elevator pitch. This allowed me to quickly explain everything the judges need to know, very breifly.

While making this website I had had to learn and use I new tool. For my project I chose to use SASS. SASS makes it quick and easy to write the CSS code for my website. CSS is the coding language that allows you to fully customize your creation.
